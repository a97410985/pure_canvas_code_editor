# code editor that implement with pure canvas(crazy)

> 純粹用canvas實現的syntax highlight code editor

功能
1. 編輯相關功能
   1. 輸入字元
   2. 選取文字
   3. 貼上剪貼簿文字
   4. 複製選取文字
   5. 壓住SHIFT鍵再按上、下、左、右鍵可以往上、下、左、右選取
2. 滑鼠
   1. 可以選取文字
   2. 選取文字時，如果滑鼠超過編輯範圍，會往上、下、左、右移動選取
   3. 可以點擊讓文字游標移動到點擊的地方
